# 📚 BookBriefApp — Поиск саммари и рейтингов книг

**BookBriefApp** — это десктопное приложение на языке C# (Windows Forms), предназначенное для быстрого поиска информации о книгах, их рейтингов и краткого содержания (саммари). 

## 🌟 Ключевые особенности проекта
- **Архитектура REST API:** Взаимодействие с глобальной базой данных *Open Library API* посредством асинхронных запросов (`HttpClient`).
- **Работа без токенов:** Приложение использует полностью открытые эндпоинты, не требующие регистрации приватных API-ключей.
- **Отказоустойчивость:** Реализован алгоритм "двойного асинхронного запроса" для извлечения скрытых описаний книг по их уникальным ключам (`bookKey`).
- **Локализация и перевод:** Внедрена автоматическая интеграция с *MyMemory Translation API* для перевода англоязычных аннотаций на русский язык «на лету» с контролем лимитов длины строки (до 450 символов).
- **Парсинг JSON:** Обработка сложных древовидных JSON-структур с помощью библиотеки `Newtonsoft.Json`.

## 🛠 Технологический стек
- **Язык:** C# (основан на принципах OOP)
- **Платформа:** .NET Framework / Windows Forms
- **Библиотеки:** Newtonsoft.Json (NuGet)
- **Инструменты контроля версий:** Git, GitHub (реализована Git-flow структура с ветками `main` и `develop`)
