Поиск саммари и рейтингов книг
приложение на языке C# (Windows Forms), предназначенное для быстрого поиска информации о книгах, их рейтингов и краткого содержания (саммари). 

## 🛠 Технологический стек
- Язык: C# (основан на принципах OOP)
- Платформа: .NET Framework / Windows Forms
- Библиотеки: Newtonsoft.Json (NuGet)
- Инструменты контроля версий: Git, GitHub (реализована Git-flow структура с ветками `main` и `develop`)
